## Kitchen Sink

A kitchen sink page showcases all available UI components and patterns in one place. Use these as reference when building new screens:

- **Full showcase**: Kitchen Sink at menu: System Administration / Kitchen Sink (`/kitchen-sink`)
- **Form controls** (text, date picker, checkbox, file upload, radio, dropdown, pill box, drop zone): Sample Form at menu: Setup / Sample Form (`/sampleform`)
- **Datatables**: See Datatable section below for smart filter, top filter, and print/export patterns
- **Miller Columns** (hierarchical drilldown): See section 2 below; demo in Kitchen Sink
- **Side Panel** (multilevel master-detail): See section 3 below; demo in Kitchen Sink
